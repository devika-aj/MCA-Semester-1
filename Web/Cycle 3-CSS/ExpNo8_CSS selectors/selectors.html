Selectors in CSS

Element selector

The Element selector selects each element by the element name

ID selector

The id selector use the id attribute to selct a particular element
The id of an element is unique within a page, so the id selctor is used to select one unique element
To selct an element within a specific id, write # character followed by the id of the element.

Class selector

The class selctor select the element with a specific class attribute
To selct an element within a specific class, write . character followed by the class name of the element.

Universal selector

The Universal selctor * selects all element element with the same style definitions.

The grouping selector selects the all the element whiht the same style definition.
